# 🌐 Sonmatreya Portfolio Website

A modern and responsive personal portfolio website built using React.js, Tailwind CSS, and Framer Motion.

## 🚀 Live Demo

🔗 Live Website: https://portfolio-3vgg.onrender.com

---

## 📌 Features

- ⚡ Modern UI Design
- 📱 Fully Responsive
- 🎨 Tailwind CSS Styling
- ✨ Framer Motion Animations
- ⌨️ Typing Effect Animation
- 📂 Projects Showcase
- 🛠 Services Section
- 📧 Contact Form Integration
- 📄 Resume Download Feature
- 🌈 Gradient Effects & Hover Animations

---

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- React Icons
- Vite
